# 2006 Revive
**Created by Viizva**

**Up to date as of February 19, 2026**

![icon](https://i.imgur.com/Dx4ndO8.png)

Accurate recreation of the 2006 interface from one of Runescape's iconic years. The 2006 time period combines the modern icons with the classic stone look a perfect mixture of past and future similar to OSRS.

- Accurate chatbox borders from 2006
- Modern icons recreated in 2006 aesthetic
- Icons positioned correctly as they were in 2006
- Kept up to date with changes published by JaGex


If there are any issues and/or suggestions feel free to message me on discord @Viizva

## Installation

1. Install the `Resource packs` plugin from the `Plugin Hub` in RuneLite.

![Resourcepacksplugin](https://i.imgur.com/8c4y8ZM.png)

2. Open the `Resource Packs` plugin settings and make sure `Don't change Interface Styles gameframe option` is enabled.

![Resourcepackssettings](https://i.imgur.com/vr0vLG2.png)

3. Enable the `Interface Styles` plugin in RuneLite and set the `Gameframe` option to `2006`.

![InterfaceStyleSetting](https://i.imgur.com/mHwPuST.png)

4. Open the `Resource packs hub` in the side panel of RuneLite, search for `2006 Revive`, and install it.

⚠️ If icons and tabs are misaligned double-check that your `Interface Styles` and `Resource packs` settings match those shown above then disable and re-enable the `Resource packs` plugin.
