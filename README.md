# Kpop Pack (by W 0 N D 3 R)

![icon](https://i.imgur.com/uTZiQEg.png)

Kpop Pack is a mostly pink dark mode pack with monochromatic muted pink and black color palette. This themed pack is inspired by Kpop elements and alt style, while keeping most assets as vanilla recolors.

Created all elements using Krita and ibisPaint X by me.

For additional photo previews from this pack, view those at this link https://imgur.com/a/3XFkhf0.
