# Victorian Rose (by W 0 N D 3 R)

![icon](https://i.imgur.com/TESqx1F.png)

Victorian Rose is a gothic victorian inspired pack with elements of roses, rabbits, and antiques. The color scheme is similar to the vanilla UI, but darker, with accents of red, gold/bronze, and sepia tone. This pack is highly customized and good for cozy gaming, while still being functional for pvm/pvp.

Created all elements using Krita and ibisPaint X by me.

For additional photo previews from this pack, view those at this link https://imgur.com/a/PSW7ktL.
