# 2005 Overhaul

![icon](https://i.imgur.com/RMojeoS.png)

An overhauled version of RuneLite’s 2005 interface, designed to recreate the original 2005 look and feel as closely as possible within the limits of OSRS.

**Created by Demonseance**

## Interface
![2005 Interface](https://i.imgur.com/JGAhosa.png)

## Installation

1. Install the `Resource packs` plugin from the `Plugin Hub` in RuneLite.

![Resourcepacksplugin](https://i.imgur.com/8c4y8ZM.png)

2. Open the `Resource Packs` plugin settings and make sure `Don't change Interface Styles gameframe option` is enabled.

![Resourcepackssettings](https://i.imgur.com/ypvTOlv.png)

3. Enable the `Interface Styles` plugin in RuneLite and set the `Gameframe` option to `2005`.

![InterfaceStyleSetting](https://i.imgur.com/3WkD8JT.png)

4. Open the `Resource packs hub` in the side panel of RuneLite, search for `2005 Overhaul`, and install it.

![Sidepanel](https://i.imgur.com/4Bt20AA.png)

> ⚠️ If icons and tabs are misaligned, double-check that your `Interface Styles` and `Resource packs` settings match those shown above, then disable and re-enable the `Resource packs` plugin.
